O que é HTML?

HTML (HyperText Markup Language) NÃO é uma linguagem de programação, é uma linguagem de marcação padrão para páginas Web, ele define o significado e a estrutura do conteúdo da web.

O HTML usa marcação para anotar texto, imagem e outros conteúdos para exibição em um navegador da web.

Ele é o esqueleto da sua página é nele que vão conter os textos e conteudo da página, depois vem o CSS e a linguagem de programação para deixar seu site mais decorado e dinamico, um complementa o ourtro. Se você aprende HTML você está pronto para fazer um site.

O que são Clients?

Clients (cliente) é a parte do sistema que está mais ligada a nós seres humanos, quando navegamos no computador entramos em um navegador como Chrome ou firefox entramos em algum site, essa é parte do cliente, pode ser considerado o navegador um app, são dispositivos ou software.

O que são Servers?

Server (servidor) é o um sistema que fornece os dados e reposta para os clientes quando solicitado, ele pode ser um computador ou um conjuntos de computadores. Ele recebe o pedido do cliente e envia a respota

Como funciona a comunicação entre cliente e servidor?

  1. O cliente envia um pedido (requisição) para o servidor por exemplo acessar um site.
  2. O servidor recebe a requisição, processa e envia.
  3. O cliente recebe a resposta ou seja o site está pronto para ser navegado.

Estrutura básica de um HTML

O HTML precisa de uma estrutura clara e bem organizada. Quando o código está bem estruturado, ele se torna mais fácil de entender, tanto para os navegadores quanto para outros desenvolvedores. Isso demonstra cuidado e profissionalismo, o que pode fazer você se destacar no mercado. Por isso, é fundamental dar sentido ao que se escreve, indo além de apenas "fazer funcionar".

"Sempre deixe o código melhor do que você o encontrou." Robert C. Martin

Diferença entre listas ordenadas e não ordenadas

A difrença entre listas ordenada e não ordenadas é que a lista ordenada ela segue uma sequência numérica, começando do número 1. Já a lista não ordenada apresenta os itens com marcadores (geralmente bolinhas), sem uma ordem específica. Essas listas são muito utilizadas em textos para organizar informações de forma clara.

Abaixo, segue um exemplo de uma lista ordenada de compras:

  1. Oléo
  2. Arroz
  3. Feijão
  4. Açucar

Não ordenada:

  • Oléo
  • Arroz
  • Feijão
  • Açucar
